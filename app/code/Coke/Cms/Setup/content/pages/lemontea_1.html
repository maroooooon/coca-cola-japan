レモンの味と香りが紅茶を引き立ててくれるレモンティーは、自宅でもおいしく作ることができます。

そこで今回は、レモンティーの基本的な作り方やおいしい飲み方を紹介します。

茶葉を使ったリーフティーやティーバッグレモンティーの作り方をはじめ、ひと味違うアレンジレシピも紹介するので、自宅でおいしいレモンティーを作りたい方はぜひ参考にしてみてください。

 

レモンティーに合う茶葉の種類

 

自宅でおいしいレモンティーを作るなら、茶葉にもこだわって選んでみてはいかがでしょうか。

レモンをはじめとする柑橘類にはリモニンという苦味成分が含まれています。

そのため、レモンティーを作るときの茶葉は、渋みの少なくフルーティーな香りの「キャンディ」、さわやかな風味とバランスの良い苦みが特徴の「ディンブラ」など、甘み、渋み、コクのバランスが絶妙なスリランカ産のセイロン紅茶がおすすめです。

また、スリランカに近いインド南部で栽培される「ニルギリ」も、セイロン紅茶に似た味わいとすっきりした香りが特徴で、レモンティーとして好まれることが多い茶葉の種類です。

 

レモンティー（ホットティー）の作り方

 

レモンティーの作り方はとても簡単です。

まずは、リーフティーやティーバッグを使ったレモンティーの基本的な作り方を紹介します。

 

リーフティーの場合

茶葉を使ってレモンティーを入れることで、より本格的な味わいを楽しむことができます。

作り方は、以下を参考にしてみてください。

材料（1人前）

  • 紅茶（茶葉）：2.5～3g　
  • 熱湯：150ml
  • レモン
  • 砂糖：好みによって適量

必要な道具

  • ティーポット
  • カップ

作り方

  1. ティーポットとカップにお湯を入れ、あらかじめ温めておきます。
  2. 温まったらお湯を捨て、ティーポットに茶葉と150mlの熱湯を入れて蓋をし、2～3分蒸らします。
  3. ポットの中をスプーンでひと混ぜしたら、紅茶をカップに注ぎます。
  4. 薄くスライスしたレモンを2～3回くぐらせたらすぐに取り出し、好みで砂糖を加えて完成です。

ティーポットの中の紅茶をカップに注ぐときは、「ベスト・ドロップ」や「ゴールデン・ドロップ」と呼ばれる最後の1滴まで残さず注ぎましょう。

また、レモンを紅茶に浸したままにすると苦味成分が出てしまうため、数回くぐらせたらレモンを取り出すのがおいしいレモンティーを作るポイントです。

 

ティーバッグの場合

ティーバッグの紅茶なら、カップに入れてお湯を注ぐだけで簡単においしいレモンティーが作れます。

材料（1人前）

  • ティーバッグ：1個
  • 熱湯：150ml
  • レモン
  • 砂糖：好みによって適量

必要な道具

  • カップ

作り方

  1. カップにお湯を入れて温めます。
  2. 温まったらお湯を捨て、ティーバッグ1個と熱湯を注いだら、皿などで蓋をして1〜1分半蒸らします。
  3. ティーバッグを軽く振って、取り出します。
  4. 薄くスライスしたレモンを2〜3回くぐらせたらすぐに取り出し、好みで砂糖を加えて完成です。

熱湯を注いだ後、カップに蓋をして蒸らすのがおいしく入れるコツです。

また、生のレモンがない場合は、レモン果汁を2滴くらい加えてもおいしいレモンティーが作れます。

 

アイスレモンティーの作り方

 

さっぱりとした味わいのアイスレモンティーも自宅で簡単に作れます。おいしいアイスレモンティーの作り方は、以下を参考にしてください。

 

リーフティーの場合

アイスレモンティーの作り方としては、温かい紅茶を作ってから冷やす方法と水出しで作る方法があります。

まずはお湯から作る方法を紹介します。

材料（1人前）

  • 紅茶（茶葉）：2.5～3g　
  • 熱湯：80ml
  • レモン
  • 氷：200g　※入りきらない場合は、グラスいっぱいまで
  • 砂糖：好みによって適量

必要な道具

  • ティーポット
  • グラス

作り方

  1. ティーポットにお湯を入れて、あらかじめ温めておきます。
  2. 温まったらお湯を捨て、ティーポットに茶葉と80mlの熱湯を入れて蓋をして、1分半〜2分蒸らします。好みで砂糖を加える場合は、茶葉をこして紅茶を別の容器に移してから入れましょう。
  3. ポットの中をスプーンでひと混ぜしたら、氷200gを入れたグラスに一気に注ぎます。
  4. カットしたレモンを軽く絞って完成です。

アイスレモンティーをおいしく作るコツは、茶葉の分量を変えずにお湯の量をホットの半分程度にすることです。

同じ茶葉の分量でもお湯を少なめにすることで、濃いめのホットティーを作りましょう。

また、温かい飲み物よりも冷たい飲み物のほうが渋味に敏感になるため、アイスの場合はホットより短めに蒸らして、渋味を抑えるのもポイントです。

甘味を加えるときは、ホットティーの状態で砂糖などを入れるか、または飲む前にガムシロップを入れてもおいしく飲めます。

 

水出しの場合

アイスレモンティーは、水出し用の茶葉を使って作ることもできます。

作り方は、以下を参考にしてください。

材料（1人前）

  • 水出し用の紅茶（茶葉）：15g
  • 水：1L
  • レモンの皮： 1／2個分
  • ガムシロップ：好みによって適量

必要な道具

  • クーラーポット

作り方

  1. クーラーポットに茶葉、水、レモンの皮を入れ、冷蔵庫で8時間程度冷やします。
  2. 茶葉とレモンの皮を取り除き、好みでガムシロップを加えて完成です。

水出し紅茶はマイルドな味わいでクセが少なく、ごくごく飲めるのが特徴です。

新鮮な茶葉と水を使うのが基本ですが、おいしく飲むためには冷蔵庫に入れて保存し、当日中に飲み切るようにしましょう。

レモンティーのおいしい飲み方はある？アレンジレシピも見てみよう


レモンティーはそのまま飲んでもおいしいですが、ほかの食材と組み合わせてアレンジするのもおすすめです。

ここからは、レモンティーのおいしい飲み方としてアレンジレシピを紹介します。

 

レモンティー×ジンジャー

レモンティーにジンジャーエールを加えると、レモンの爽やかさと炭酸のはじけるおいしさが加わって、いつもとは違うレモンティーに仕上がります。

ジンジャーレモンティーの作り方は、アイスレモンティーとジンジャーエールを氷の入ったグラスに1：1の割合で入れるだけ。簡単でおいしい飲み方なので、ぜひ試してみてください。

 

レモンティー×はちみつ

甘いレモンティーが好みの場合は、レモンティーとはちみつの組み合わせをぜひ試してみてください。レモンとはちみつは相性抜群です。

ホットの場合は、最後に入れる砂糖をはちみつに変えるだけでおいしいはちみつレモンティーが完成します。

また、アイスレモンティーにはちみつを加える場合は、紅茶が熱い状態のときにはちみつを加えるのがポイントです。

 

手軽においしいレモンティーが飲める！ペットボトルも活用しよう

 

レモンティーの作り方はとても簡単ですが、もっと手軽にレモンティーを楽しみたいという場合は、ペットボトルのレモンティーを活用しましょう。

ペットボトルで販売されているレモンティーも、茶葉やレモンにこだわって作られたものが多いため、手軽においしいレモンティーを飲むことができます。

また、アレンジレシピも簡単にできるので、ジンジャーエールを合わせてシュワっと爽やかなアイスレモンティーを作ったり、フルーツと合わせてちょっと贅沢なフルーツティーを作ったりするのもおすすめです。

ペットボトルのレモンティーは、ホット・アイスどちらも販売されています。飲料メーカーによって茶葉や製法が異なるため、好みに合う味や香りのレモンティーを探してみましょう。

今回はレモンティーの基本的な作り方、おいしく作るコツやおすすめの飲み方を紹介しました。

レモンティーは、自宅で簡単に作ることができ、ほかの食材と組み合わせることでさまざまな飲み方ができます。

また、茶葉やレモンにこだわって作られたペットボトルのレモンティーなら、ジンジャーエールと合わせたり、ほかのフルーツを入れたフルーツティーにしたりなど、おしゃれで贅沢なレモンティーも気軽に楽しむことができるでしょう。
