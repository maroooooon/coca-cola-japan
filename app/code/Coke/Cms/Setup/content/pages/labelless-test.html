近年、ラベルがないペットボトル、いわゆる「ラベルレスボトル」を販売する飲料メーカーが増えています。

そこで今回は、ラベルレスボトルがなぜ急速に広がりを見せているのか、人気の理由を紹介します。

ラベルレスボトルが増加中！いつから始まった？

 

ラベルレスボトルとは、ラベルを付けずに販売するペットボトル飲料のことです。

資源有効利用促進法の一部改正を機に、2020年頃から飲料メーカー各社がラベルレスボトルの販売を始めました。

通常のペットボトル製品は、ラベルに原材料名などの表示をすることが義務付けられていますが、ラベルレスボトルは法定表示を外装ダンボールやケースに記載することにより、ラベルを付けていなくても販売可能となっています。

ラベルレスボトルは、ネット通販などでペットボトル飲料を箱買いすることを想定して販売されているもので、コンビニなどの店頭で販売されるケースはまだ少ないのが現状ですが、最近では、外出自粛やテレワークによりネット通販の利用者が増えたことで、急速に需要が拡大しています。

ラベルレスボトルはなぜ人気がある？

 

ラベルレスボトルが人気を集めている理由としては、次のようなことが挙げられます。

 

ごみの分別・リサイクルが楽にできる

 石油を原料として作られたペットボトルは、正しく回収してリサイクルすることで、リサイクルペットボトルをはじめとする新たな製品へと生まれ変わることができます。

ペットボトルをリサイクルごみとして排出するには、素材が異なるラベルをあらかじめ剥がしておく必要がありますが、1本ずつ剥がしてごみを分別する作業は、手間がかかり面倒だと感じる方も多いでしょう。

しかし、ラベルレスボトルならその手間がなくなるので、リサイクルしやすいとされています。

実際にラベルレスボトルを購入した方からも、「ラベルを剥がす手間が省けるので助かる」「プラスチックごみの量が減った」といった声が多く挙がっているようです。

 

サステナブルな考え方に多くの人が共感

 サステナブルには「持続可能な」という意味があり、資源を大切にし、再利用できる資源は循環利用することで、将来にわたって持続可能な社会を目指す取り組みのことです。

そんなサステナブルな考え方に多くの方が共感していることも、ラベルレスボトルが注目される理由の一つとなっています。

ペットボトルのラベルにはプラスチックが使われているため、ラベルレスボトルにすることでプラスチック資源の節約につながります。

また、ラベルレスボトルはキャップを外すだけで分別しやすく、適切にリサイクル処理を行うことで、何度でも新たなペットボトルに生まれ変わることができるため、「ペットボトルはごみではなく資源である」という見方が広がっているのです。

 

「ボトルtoボトル」という取り組みも

 リサイクルについては、ラベルレスだけでなく「ボトルtoボトル」という考え方も重要です。

これは、使用済みペットボトルを回収して新たなペットボトルとして再生させる取り組みを指します。

日本コカ･コーラ株式会社では、2022年までにリサイクルPET樹脂使用率50％、2030年までにペットボトルへの新たな石油由来原料の使用をゼロとすることなどを目指しています。

リサイクルPET樹脂を100％使用した容器のラベルには「100％リサイクルペット」の表示が記載されているので、ラベルレスボトルの販売がない店頭などでは、こういった製品を選ぶことで持続可能な社会に貢献することができるのです。

 

見た目がおしゃれ

 ラベルがないことで見た目がすっきりするのはもちろん、デザイン性の高いラベルレスボトルが販売されるようになったことも、ラベルレスボトルの人気を後押ししています。

ペットボトルのラベルには、食品表示法で定められた原材料などの法定表示をするだけでなく、ラベルのデザインによって商品の魅力をアピールするという役割もあります。

そのため、ラベルを付けないラベルレスボトルの場合は、ラベルではなくペットボトル自体のデザイン性に力を入れる飲料メーカーが増えているのです。

水やお茶、炭酸飲料でも！ラベルレスボトルでひと手間を軽減

 

ラベルレスボトルは、資源有効利用促進法の一部改正を機に、2020年頃から徐々に浸透してきました。

ごみの分別をするときに1本ずつラベルを剥がす手間を省くことができるほか、プラスチック資源の削減にもつながります。

ラベルレスボトルは、水やお茶、炭酸飲料など種類も豊富に展開されており、ネット通販を通して気軽に購入できます。

リサイクルする際にラベルを剥がす手間がかからないですし、すぐに始められるサステナブルな行動として、普段の生活にラベルレスボトルを取り入れてみてはいかがでしょうか。
