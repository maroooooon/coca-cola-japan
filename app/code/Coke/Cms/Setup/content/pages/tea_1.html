紅茶の入れ方にはさまざまなタイプがあり、茶葉を抽出したり、ティーバッグにお湯を注いだり、水出しをしたり、ミルクを入れたりなど、入れ方によって違った味わいを楽しめます。

そこで今回は、紅茶の入れ方や手順、おいしく入れるコツについて、紅茶のタイプ別に解説していきます。

 

紅茶の入れ方【リーフティー編】

 

リーフティーとは、茶葉をそのままポットに入れて作る紅茶のことで、最も一般的な紅茶の入れ方です。

手順はとても簡単ですが、「ゴールデンルール」と呼ばれるおいしい入れ方を知っておくと、より本格的な紅茶の味を楽しめます。

 

用意するもの

  • やかん
  • くみたての水道水：150ml
  • お湯
  • ティーポット
  • カップ
  • 紅茶（茶葉）：2.5～3g　
    ※大きめの茶葉なら3～4g

手順

  1. ティーポットとカップにお湯を入れ、あらかじめ温めておきます。
  2. 温まったらお湯を捨て、ティーポットに茶葉を入れましょう。
  3. 新鮮なくみたての水道水150mlをやかんに入れて沸騰させます。
    ※大きめの泡が勢いよく出てくるまで
  4. 沸騰直後のお湯をティーポットに注ぎ、蓋をして2〜3分蒸らします。
    ※大きめの茶葉なら3～4分
  5. ティーポットの中をスプーンなどでひと混ぜしたら、紅茶をカップに注ぎます。

おいしく入れるポイント

  • ティーポットとカップを温めておく
  • 質の良い茶葉を使う
  • 茶葉の分量を守る
  • くみたての水道水を使う
  • 沸騰直後のお湯を使う
  • 茶葉をしっかりと蒸らす

この6つのポイントが、紅茶を入れるときのゴールデンルールです。

日本で市販されている紅茶の茶葉は、そのほとんどが上質ですが、せっかく質の良い茶葉を使ったとしても、古いものや傷んでいるものではおいしい紅茶は入れられません。

また、茶葉の量が少なすぎると香りや風味に物足りなさを感じることになり、多すぎると渋みが強くなるため、適切な分量を守りましょう。

紅茶を入れる際、「水道水ではなくミネラルウォーターのほうが良いのでは？」と思う方もいるかもしれませんが、日本の水道水のほとんどが紅茶に合う軟水であり、さらに水道水には空気が含まれているため、しっかりと沸騰させてティーポットに注ぐことでジャンピング（茶葉の上下運動）が起こりやすくなります。

これによって茶葉がきれいに開くため、香りや味わいの深い紅茶を入れることができるのです。

ティーポットとカップは事前に温めておき、沸騰直後のお湯の温度を下げないように注意しましょう。

紅茶の茶葉は、ただ長時間蒸らせば良いということではなく、蒸らしすぎると渋みが出る原因になるため、紅茶の種類によってどれくらいの蒸らし時間が適切なのかを確認しておくことが大切です。

紅茶の入れ方【ティーバッグ編】

 

紅茶のティーバッグは、お湯を注ぐだけで簡単に紅茶を入れることができるため、その手軽さが何よりも人気のポイントです。

ティーバッグで入れる紅茶も、手順やポイントを押さえておくことでおいしく仕上がります。

 

用意するもの

  • やかん
  • くみたての水道水：150ml
  • お湯
  • ティーバッグ：1個
  • カップ
  • ソーサー（小皿でも可）

手順

  1. くみたての水道水をやかんに入れて沸かします。
  2. カップにお湯を入れて、温まったらお湯を捨てます。
  3. 水道水が沸騰したら、ティーカップに注ぎます。
  4. ティーバッグ1個をカップのふちからすべり込ませるようにして、そっと入れます。
  5. ソーサー（小皿でも可）を蓋にして、1〜1分半蒸らします。
  6. ティーバッグを2〜3回軽く振ったら、カップから取り出しましょう。

おいしく入れるポイント

  • カップはあらかじめ温めておく
  • 1杯の紅茶に1個のティーバッグ
  • 沸騰直後のお湯を使う
  • ティーバッグはそっと取り出す

ティーバッグには、1杯分の紅茶がおいしくできるように計量された茶葉が入っているため、紅茶1杯につき1個のティーバッグを使いましょう。

カップを事前に温めておくことで、沸騰したお湯を冷ますことなく紅茶を入れられます。

ティーバッグを蒸らし終わったら、渋みが出ないようにそっとカップから取り出すのが、紅茶をおいしく入れるコツです。

水出し紅茶（アイスティー）の入れ方

 

水出し紅茶とは、冷水で抽出した紅茶のことを指し、沸騰直後のお湯で抽出するホットの紅茶と比べて、苦みや渋みが少なくまろやかな口当たりが特徴です。

続いては、水出し紅茶のおいしい入れ方を紹介します。

 

用意するもの

  • 蓋付きポット
  • 冷水：1L
  • 紅茶（茶葉）：12g　
    ※大きめの茶葉なら15g
  • 氷：4～5個

手順

  1. 蓋付きのポットに、茶葉を入れます。
  2. ポットに氷を4〜5個入れ、冷水1Lを注ぎます。
  3. 冷蔵庫で8時間ほど冷やして抽出させます。
  4. 茶葉を取り出したら、完成です。

おいしく入れるポイント

  • 新鮮で良質な茶葉を使う
  • 抽出温度を下げる

茶葉本来の味わいがじっくりと抽出される水出し紅茶を入れるときは、新鮮で良質な茶葉を選びましょう。

また、氷を入れて抽出する温度を下げることでより透明感が増し、すっきりとした味わいになります。

夜に作っておくことで、朝にはおいしい水出し紅茶が飲めるので、ぜひ試してみてください。

ミルクティーの入れ方

 

お湯で抽出した紅茶にミルクを入れて飲むのがミルクティーです。

ミルクティーは紅茶の本場イギリスで最も好まれている飲み方で、紅茶の渋みを抑えたマイルドな味わいが特徴です。

続いては、おいしいミルクティーの入れ方を紹介します。ミルクティーも入れる手順とポイントを押さえて、本格的な味を楽しみましょう。

 

用意するもの

  • やかん
  • くみたての水道水：150ml
  • お湯
  • ティーポット
  • カップ
  • 紅茶（茶葉）：2.5～3g　
    ※大きめの茶葉なら3～4g
  • 常温のミルク：20〜30ml

手順

  1. ティーポットとカップにお湯を入れ、あらかじめ温めておきます。
  2. 温まったらお湯を捨て、ティーポットに茶葉を入れましょう。
  3. 新鮮なくみたての水道水150mlをやかんに入れて沸騰させます。
    ※大きめの泡が勢いよく出てくるまで
  4. 沸騰直後のお湯をティーポットに注ぎ、蓋をして2〜3分蒸らします。
    ※大きめの茶葉なら3～4分
  5. ティーポットの中をスプーンなどでひと混ぜしたら、紅茶をカップに注ぎます。
  6. 常温のミルク20〜30mlをカップに入れ、軽く混ぜましょう。

おいしく入れるポイント

  • ティーポットとカップを温めておく
  • 質の良い茶葉を使う
  • 茶葉の分量を守る
  • くみたての水道水を使う
  • 沸騰直後のお湯を使う
  • 茶葉をしっかりと蒸らす
  • 常温のミルクを使う

おいしいミルクティーを作るためには、まず紅茶を正しく入れることが大切です。ゴールデンルールを思い出して、おいしい紅茶を入れましょう。

また、ミルクを加熱すると臭みの原因になり、冷たいミルクを使うと紅茶の温度を下げてしまうので、常温のミルクを使いましょう。

紅茶を簡単においしく飲む方法

 

おいしい紅茶をもっと簡単に入れて飲みたい方には、ペットボトルの紅茶もおすすめです。

自分でおいしい紅茶を入れるとなると、ティーポットやカップをあらかじめ温めておいたり、くみたての水道水をしっかりと沸騰させたり、茶葉を蒸らす時間も考慮しなければならないため、手間と時間がかかります。

ペットボトルの紅茶なら、いつでもどこでも手軽においしい紅茶を飲むことができ、さらにストレートティーやレモンティー、ミルクティーやフルーツティーなど種類も豊富にあります。

こだわりの茶葉と製法で作られたものが多いため、自分好みの紅茶を探しやすいでしょう。紅茶をもっと簡単においしく飲みたいという方は、ぜひペットボトルの紅茶も試してみてください。

 

紅茶は入れ方も大事！簡単でおいしいペットボトルも活用しよう

 

自宅でおいしい紅茶を楽しむためには、入れ方とコツを押さえることが大切です。

ちょっとした手間をかけることで、いつもの紅茶がよりおいしくなるため、今回紹介したゴールデンルールや手順、おいしく入れるポイントを参考にしながら、ぜひ挑戦してみてはいかがでしょうか。

また、簡単においしい紅茶が飲みたい、外出先で気軽に紅茶を楽しみたいという方は、ペットボトルの紅茶も活用してみてください。
